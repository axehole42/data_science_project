# data_science_project
This repo is meant to be used for the preparation, training testing and validation of our models for the introduction to AT in finance project.

To upload push and pull, please download and set up GIT within vscode.

https://git-scm.com/install/windows

To clone this repo, enter the following command:

git clone https://github.com/axehole42/data_science_project

You will have to log in prior to pushing.

To update the repo, please enter the following command:

git add . 
git commit -m "Comment about changes here"
git push origin main
